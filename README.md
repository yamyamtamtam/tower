# Tower Defense

ブラウザで動くアイソメトリック(クォータービュー)タワーディフェンスゲーム。
ビルド不要・依存ライブラリなしの素のHTML/CSS/JavaScriptで動作します。

## 遊び方

`index.html` をブラウザで開くだけで起動します。

```
# ローカルサーバーを使う場合(任意)
python3 -m http.server 8000
# → http://localhost:8000
```

- 下部パネルからタワーを選び、空きマスをクリックして建設(キー `1`〜`4` でも選択可)
- 「Wave 開始」ボタン(またはスペースキー)で敵の侵攻開始
- 建設済みタワーをクリックすると強化・売却が可能
- 右クリック / Esc で選択解除
- 全30 Waveを防衛したらクリア

### タワー

| タワー | 特徴 |
|---|---|
| 🏹 アロー | 安価で連射が速い。序盤の主力 |
| 💣 キャノン | 範囲爆発ダメージ。集団に強い |
| ❄️ アイス | 敵を減速させる支援タワー |
| 🎯 スナイパー | 長射程・高火力・低連射。タンクやボス向け |

## 画像(3D風スプライト)の差し替え

現在はコード描画による擬似3D表示ですが、`assets/img/` に画像を置いて
`js/assets.js` の `ASSET_MANIFEST` に登録すると自動で画像描画に切り替わります。
詳細は [assets/img/README.md](assets/img/README.md) を参照してください。

## ファイル構成

```
index.html      … エントリーポイント(HUD等のUIもここ)
style.css       … UIスタイル
js/config.js    … ゲームバランス・マップ・Wave定義
js/assets.js    … 画像アセット管理(差し替えレイヤー)
js/game.js      … ゲーム本体(ロジック+描画)
assets/img/     … 画像置き場
```
